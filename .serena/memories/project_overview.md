Purpose: Solana memecoin trading bot with backend trading engine plus Next.js dashboard. Stack: TypeScript ESM on Node.js, Express API, Prisma 7 with PostgreSQL, BullMQ/Redis, Next.js 16 dashboard, Solana integrations via Helius/Birdeye/Jupiter/Jito. Main working area is trading_bot/. Key safety rules: use logger utility instead of console, preserve circuit breaker/retry/rate-limit patterns around external APIs, prefer minimal reversible changes, and be especially careful in src/core, src/strategies, src/services, and prisma. Backend entrypoint is trading_bot/src/index.ts; API server in src/api/server.ts; dashboard in trading_bot/dashboard/.
