# Market Memory

Store durable market-regime notes that affect how the bot should be interpreted.

- Regimes where discovery quality changes sharply
- Venue-specific behavior that distorts metrics
- Time-of-day effects
- Recurring liquidity traps
- Conditions that make recent backtests or dry runs misleading
